2. Marcação Rápida (Botão Direito)

A forma mais rápida de usar a extensão. Basta ir para a página inicial do processo digital e clicar com o botão direito em um processo. As Tags Rápidas que você configurou (na Seção 1) aparecerão!

Tutorial e Benefícios:

  • Ação Principal: Clique com o Botão Direito no item da lista de processos.
  • Escolha uma Tag Rápida no menu.
  • Velocidade: Aplicação instantânea sem abrir janelas.
  • Teste: Clique em qualquer tag ao lado para simular a aplicação instantânea.

3. Popup: Criação Manual de Tags

Use o Popup (o ícone da extensão) para criar uma tag com nome e cor personalizados, uma a uma. Isso é útil para tags únicas que não precisam ser salvas como Tags Rápidas.

Como Funciona a Criação Manual:

  • Permite customizar Nome e Cor antes de salvar.
  • O ID do processo é preenchido automaticamente.
  • A tag é salva no histórico do processo.
